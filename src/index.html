Hash Sort

An algorithm that I played around with in an attempt to take advantage of the O(1) time complexity of hashing to derive a method of sorting. For large data sets that are normally distributed, I found that this method of sorting could be more efficient than a simple quick sort algorithm. The code that I wrote can be found on GitHub.
