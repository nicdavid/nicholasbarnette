Iconly

http://nicholasbarnette.com/iconly

I started this project in order to be able to easily integrate SVG images into my applications. I have created an icon font that anyone is able to import into their HTML pages. The site has the ability to browse the available icons and will soon allow for people to request custom icons.

Overview

I wanted a way to import my set of icons into multiple different applications. Since I am reusing them, I wanted a way to update the icons in one location and for the updates to propogate through all of my other applications. In the end, I decided that I would just create an icon font and allow anyone and everyone to use them.

To be able to use these icons, one will just need to import a single JavaScript file into the head tag of their page. This technique allows me to have control over what CSS files people will be pulling in, so if I ever did want to give more or less access to certain people, I would be able to do so without breaking the import for everyone.
